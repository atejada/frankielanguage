About Frankie

Frankie is a lightweight programming language created for experimentation, learning, and fun. Stitched together from Ruby • Python • R • Fortran

The goal of Frankie is to be:

  • Simple to read
  • Easy to modify
  • Small enough to understand completely
  • Friendly for language experimentation

Hello World

print("Hello, Frankie!")

Example

Variables

name = "Frankie"

fn greet(name) {

  print("Big number")

Getting Started

Clone the repository:

git clone https://github.com/atejada/Frankie
cd Frankie

Build the interpreter:

make

Run a program:

frankie hello.fr
