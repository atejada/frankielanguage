# Frankie Programming Language

A small, friendly, experimental programming language.

--- ## About Frankie Frankie is a lightweight programming language created for experimentation, learning, and fun Stitched together from Ruby • Python • R • Fortran The goal of Frankie is to be: * Simple to read * Easy to modify * Small enough to understand completely * Friendly for language experimentation --- ## Hello World ```frankie print("Hello, Frankie!") ``` --- ## Example ### Variables ```frankie name = "Frankie" age = 5 ``` ### Functions ```frankie fn greet(name) { print("Hello " + name) } ``` ### Conditionals ```frankie if x > 10 { print("Big number") } ``` --- ## Getting Started Clone the repository: ``` git clone https://github.com/atejada/Frankie cd Frankie ``` Build the interpreter: ``` make ``` Run a program: ``` frankie hello.fr ``` --- ## Philosophy Frankie follows a few simple ideas: * Programming languages should be easy to read. * Implementations should be understandable. * Experimentation should be encouraged. --- ## Contributing Contributions are welcome!
